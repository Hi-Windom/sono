# v3.x 高频噪声超标问题修复计划（HiFi 版）

## 设计理念

**不妥协音质**：不降低任何参数值，不降低母带临场提升，保持算法全力工作。

**HiFi 手段**：用** Linkwitz-Riley 分频 + gentle 衰减**替代** Butterworth 陡峭低通**。分频器是 HiFi 领域的标准工具，相位相干、叠加平坦，比低通滤波器音乐化得多。

---

## 修改方案（4 项改动）

### 改动 1: 重写 `_hf_protect` — 分频器 + gentle 衰减（所有 4 个算法）

**当前**: 6 阶 Butterworth 6kHz 低通 — 陡峭滚降，相位偏移大，不 HiFi

**改为**: 2 阶 Butterworth 7kHz 分频 + 高频带 gentle 衰减

```python
def _hf_protect(y, sr):
    nyq = sr / 2
    crossover = 7000  # 7kHz 分频点
    if crossover >= nyq:
        return y
    # 2 阶 Butterworth 分频（LR2 风格，叠加后平坦）
    sos_low = butter(2, crossover / nyq, btype='low', output='sos')
    sos_high = butter(2, crossover / nyq, btype='high', output='sos')
    low = sosfiltfilt(sos_low, y, axis=-1)
    high = sosfiltfilt(sos_high, y, axis=-1)
    # 高频带 gentle -4dB 衰减
    return low + high * 0.63
```

**为什么更 HiFi**:
| 特性 | 旧方案 (6阶6kHz低通) | 新方案 (2阶7kHz分频) |
|------|---------------------|---------------------|
| 0-5kHz | 几乎无损 | 完全无损 |
| 5-7kHz | 严重衰减 (-6dB~-18dB) | 几乎无损 (<0.5dB) |
| 7kHz 处 | -3dB | -3dB (分频点) |
| 8kHz 以上 | -18dB~-36dB | -4dB gentle 衰减 |
| 相位 | 严重偏移 | 最小偏移 |
| 音乐性 | 差（"盖被子"效应） | 好（自然的空气感控制） |

**影响**: 保留 0-7kHz 全部细节（人声主体、齿音、临场感），仅 gentle 减少 7kHz+ 能量。

### 改动 2: v3.1 `_vocal_exciter` 三次饱和 → tanh 饱和

**理由**: 三次饱和 `1.5*wet - 0.5*wet**3` 产生丰富奇次谐波（3次、5次、7次...）。tanh 是更音乐化的饱和曲线，高次谐波自然滚降。

**当前** (`repair_v3_1/core.py` line 357-359):
```python
wet = dry * 1.5
wet = np.clip(wet, -1.0, 1.0)
wet = 1.5 * wet - 0.5 * wet**3
```

**改为**:
```python
wet = np.tanh(dry * 2.0) * 0.7
```

**为什么更 HiFi**: tanh 是模拟硬件（电子管、磁带）的自然饱和模型，谐波结构更温暖自然。三次饱和是数学构造，高次谐波不自然。

---

### 改动 3: v3.0/v3.0a 添加母带处理阶段

**理由**: v3.0/v3.0a 当前无母带处理，添加母带提升音质完整性。

**v3.0**: `mix_tracks` → `_mastering_standard`(presence 0.15) → `_hf_protect` → `_soft_peak_limit`
**v3.0a**: `mix_tracks` → `_mastering_standard_lite`(presence 0.15) → `_hf_protect` → `_soft_peak_limit`

presence boost 保持 0.15（与 v3.1 一致），不降低。

---

### 改动 4: 母带处理后添加 `_hf_protect`（所有 4 个算法）

**理由**: 母带处理在混音后添加临场提升，这部分未被 `process_vocal_track` 中的 `_hf_protect` 保护。在母带后添加 `_hf_protect`（使用新的分频器方案），作为安全网。
